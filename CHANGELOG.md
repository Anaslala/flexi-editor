# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-01-31

### Added
- Initial release
- Core editor functionality
- Plugin architecture
- 30+ built-in plugins
- Responsive design
- Accessibility features
- Full documentation
- React and Vue examples

### Features
- Text formatting (Bold, Italic, Underline, Strikethrough)
- Headings (H1-H6)
- Lists (Ordered, Unordered)
- Text alignment
- Links and images
- Tables
- Code blocks
- Colors and fonts
- Undo/Redo
- Find & Replace
- Word count
- Fullscreen mode
- Print support
- Drag & drop images
- Customizable toolbar
- Theme support
- Mobile responsive
