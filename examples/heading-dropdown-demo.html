How to Use the Heading Dropdown:

  • Select Text: Highlight any text in the editor
  • Choose Heading: Click the "Heading" dropdown in the toolbar
  • Pick Level: Select from Paragraph, H1, H2, H3, H4, H5, or H6
  • Auto-Update: The dropdown shows the current heading level
